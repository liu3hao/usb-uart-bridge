# Compact CH340 USB-C USB-UART bridge
This repository contains the files for a compact USB-C to UART bridge using the CH340 chip. The main purpose is to create a compact dev board that can be used for UART communications.

## Project files
1. Schematics - designed with [circuitscript](https://circuitscript.net)
2. PCB layout - designed with [KiCad](https://www.kicad.org/)

## Production
The initial production was done with [JLCPCB](https://jlcpcb.com/) and parts are completely sourced from LCSC.
